# Demo Script

**Target length:** 90-120 seconds
**Format:** Screen recording with voiceover
**Tagline:** "AgentCFO — an autonomous financial agent for startups, built on Hermes, NVIDIA NemoClaw, and Stripe"

## Scene 1: The Problem (15s)
*Screen: Empty dashboard, then a messy spreadsheet*

> "Every startup founder spends a third of their time on finance they're not trained for. They track burn rate in spreadsheets, chase invoices manually, and make spending decisions on gut feeling. Hiring a CFO costs $15K a month. AgentCFO gives them one for free."

## Scene 2: The Agent Wakes Up (15s)
*Screen: Dashboard loading, data flowing in from Stripe*

> "AgentCFO is a Hermes agent running inside an NVIDIA NemoClaw sandbox. It connects to your Stripe account and starts analyzing your financial data immediately. Here it's pulling in three months of transaction history..."

*Show: Transactions appearing, categories being assigned in real-time*

## Scene 3: The Agent Thinks (20s)
*Screen: Agent activity feed showing reasoning*

> "It doesn't just categorize — it reasons. Watch it analyze our burn rate..."

*Show: Agent decision entry — "Detected 23% increase in infrastructure spend over past 30 days. Cloud hosting costs grew from $2,100 to $2,583. Recommend reviewing instance sizing or negotiating reserved pricing. Projected annual impact: +$5,796."*

> "That's not a dashboard metric. That's a financial opinion."

## Scene 4: The Agent Acts (25s)
*Screen: Expense approval flow, budget enforcement*

> "When someone submits an expense, the agent checks it against your budget and policy. Here's a $3,000 marketing tool request..."

*Show: Agent decision — "Evaluating expense request: $3,000/yr marketing analytics tool. Marketing budget remaining this month: $4,200. Current tool stack includes [similar tool]. Recommend: DENY — duplicate capability. If approved, would consume 71% of remaining monthly marketing budget."*

> "The agent doesn't just approve everything. It has opinions and it explains them."

## Scene 5: The Payoff (20s)
*Screen: Cash flow forecast, runway visualization*

> "And the reason it all matters — runway. The agent continuously forecasts your cash position based on actual revenue trends and committed expenses. Right now it's showing 8.3 months of runway, with a warning that drops to 6.1 months if our current hiring plan goes through."

*Show: Forecast chart, scenario comparison*

> "All of this runs in a sandboxed NVIDIA NemoClaw environment with network policies that prevent the agent from exceeding its authority. Every decision has an audit trail. It's not just smart — it's safe."

## Scene 6: Close (10s)
*Screen: Full dashboard overview*

> "AgentCFO. Built on Hermes, NVIDIA NemoClaw, and Stripe. An autonomous financial brain for every startup that can't afford a real one yet."

*Logo + links*

## Recording Notes

- Use Stripe **test mode** with realistic data (not "Test Customer 1")
- Pre-load 3 months of transaction history
- Ensure agent responses are fast (< 5s) — pre-cache if needed
- Dashboard should be dark theme (looks better on video)
- Use OBS or similar for screen capture
- Record at 1080p minimum
